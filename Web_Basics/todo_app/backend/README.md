# Quickstart


Folgende befehle immer im backend ordner ausführen:

---

Beim ersten mal wenn das Projekt geklont wird:

```bash
python3 -m venv .venv
```

---

Zu beginn jeder Session:

```bash
source .venv/bin/activate

pip install -r requirements.txt
```

---

Zum starten des Servers:

```bash
fastapi dev main.py

# Der Server läuft dann unter http://localhost:8000
```

---

Beenden des Servers:

```bash
# Tastenkombination: control + C
```
